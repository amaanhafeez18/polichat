Polichat HR Assistant

How to Use

To use Polichat HR Assistant, start by opening the chat and saying "Hi" or asking a specific HR-related question. The assistant will provide you with the information or guide you through the process step-by-step.

How to Reset

If you need to reset the conversation or start over, simply type "reset" or "start over". The assistant will clear the current context and begin a new session.

How to Return to Menu

To return to the main menu at any point, type "menu" or "main menu". This will bring up the list of options available for you to choose from.
